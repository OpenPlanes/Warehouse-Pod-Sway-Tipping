# Warehouse-Pod-Sway-Tipping
Comparative structural study of a robotic storage pod center support. Modeled rectangular and cylindrical welded designs in SolidWorks and analyzed stress distribution, weld load paths, and deformation using ANSYS FEA. Includes Python simulation exploring how structural changes influence pod stability and tipping behavior.
